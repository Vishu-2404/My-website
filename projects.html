High-Speed SLA 3D Printer Resin Vat System

Designed and optimized an innovative resin vat system for high-speed SLA 3D printing, solving critical challenges in beam intensity uniformity and material flow dynamics.

3D Printing SOLIDWORKS Manufacturing Fluid Dynamics

Project Challenge

High-speed micro-SLA 3D printing faced significant challenges with uneven laser intensity distribution and resin leakage, limiting print quality and production efficiency. Traditional vat designs couldn't handle the increased processing speeds.

Innovation & Design

  • Designed 15+ resin-vat-tank components using SOLIDWORKS
  • Integrated 3-point leveling system for uniform laser intensity
  • Implemented PTFE O-ring sealing to prevent resin leakage
  • Optimized resin flow patterns for consistent material distribution
  • Created modular design for easy maintenance and replacement

Technical Solutions

The innovative design addressed beam intensity variations through precision-engineered leveling mechanisms while the advanced sealing system eliminated material waste. The modular approach enabled rapid prototyping and testing iterations.

Results & Impact

Achieved 90% improvement in beam intensity uniformity and eliminated resin leakage issues. The design enables consistent high-quality prints at increased processing speeds, contributing to more efficient additive manufacturing workflows.

Developed an accessible web application that converts morse code input to synthesized speech, empowering aphonic individuals with an innovative communication tool.

Python Flask Accessibility Speech Synthesis

Social Impact Mission

This project addresses the communication challenges faced by aphonic individuals by providing an intuitive, technology-based solution that converts simple morse code inputs into clear synthesized speech.

Technical Implementation

  • Developed Flask web application with responsive user interface
  • Implemented space-bar morse code input system for accessibility
  • Created 3-phase Text-to-Speech (TTS) pipeline for natural speech synthesis
  • Integrated real-time morse code translation algorithms
  • Designed user-friendly interface for individuals with limited mobility

User-Centered Design
